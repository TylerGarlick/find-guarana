# find-guarana

Find Guarana beverages at nearby stores, restaurants, and vendors.

## Mission
Help users locate Guarana (the delicious Brazilian soft drink) near them.

## Features
- 📍 Location-based search for nearby vendors
- 🏪 Find Guarana at stores, convenience shops, restaurants
- 📱 Get directions to the nearest location
- ⭐ Save your favorite vendors

## Development

```bash
npm install
npm run dev
```
